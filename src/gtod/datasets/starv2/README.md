# STARv2 Dataset

This directory contains the STARv2 dataset, introduced as part of the AnyTOD paper [Zhao et. al, 2022](https://arxiv.org/abs/2212.09939). The STARv2 dataset is an updated version of the [original STAR dataset](https://github.com/RasaHQ/STAR) ([Mosig et. al, 2020](https://arxiv.org/abs/2010.11853)) with richer annotations provided by the D3ST model ([Zhao et. al, 2022](https://arxiv.org/abs/2201.08904)).
